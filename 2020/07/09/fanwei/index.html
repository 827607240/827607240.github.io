实现自动化签到

前言

在OA上签到打卡和签退，但是总有些时候会忘记打卡，每天签到签退又很麻烦，于是就想到通过脚本实现自动化签到签退。

获取请求包信息

首先打开网页打开F12，点击Network，点击签到。会出现一些请求文件，打开并查看Headers，查看preview显示得是返回的信息。

请求包内容，request headers

'Content-Type': 'application/x-www-form-urlencoded',
'Referer': '从哪个网页跳转过来的',
'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/83.0.4103.116 Safari/537.36',
'Cookie': ''

查看是GET请求还是POST请求。

签到脚本

import requests
import time

url = 'http://xxx/hrm/schedule/HrmScheduleSignXMLHTTP.jsp?t=0.2655744879360351'
headers = {
    'Content-Type': 'application/x-www-form-urlencoded',
    'Referer': 'http://xxx/wui/main.jsp',
    'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/83.0.4103.116 Safari/537.36',
    'Cookie': 'xxx'
}
response = requests.post(url=url,headers=headers)
print(response.text)

签到返回内容

签到成功。

添加日志功能，只记录了运行时间。这个脚本还不够完美。

import requests
import time
import re

url = 'http://xxx/hrm/schedule/HrmScheduleSignXMLHTTP.jsp?t=0.2655744879360351'
headers = {
    'Content-Type': 'application/x-www-form-urlencoded',
    'Referer': 'http://xxx/wui/main.jsp',
    'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/83.0.4103.116 Safari/537.36',
    'Cookie': ''
}
response = requests.post(url=url,headers=headers)

now = time.strftime('%Y-%m-%d %H:%M:%S',time.localtime())
msg = response.text
time = re.findall(r'<tr>(.+?)</tr>',msg)
print(time)
try:
    with open('/home/admin/oa/log/log.txt','a') as f:
        f.write('{} --- {}\n'.format(now,time))
        f.close()
        print(response.text)
except Exception as e:
    print(e)

运行成功后，测试。

设置定时任务

编辑/etc/crontab文件。

30 8 * * 1-5 root sh /home/admin/oa/oa.sh

每周一到周五的8点半运行这个脚本。

查看记录的日志文件。

自动签到部分大功告成。

自动签退

点击签退，抓包，获得签退时的请求链接。

修改脚本请求的链接即可完成签退脚本。

import requests
import time
import re

url = 'http://xxx/hrm/schedule/HrmScheduleSignXMLHTTP.jsp?t=0.961868599700483'
headers = {
    'Content-Type': 'application/x-www-form-urlencoded',
    'Referer': 'http://xxx/wui/main.jsp',
    'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/83.0.4103.116 Safari/537.36',
    'Cookie': 'xxx'
}
response = requests.post(url=url,headers=headers)

now = time.strftime('%Y-%m-%d %H:%M:%S',time.localtime())
msg = response.text
time = re.findall(r'<tr>(.+?)</tr>',msg)
print(time)
try:
    with open('/home/admin/oa/log/log.txt','a') as f:
        f.write('{} ---quit success {}\n'.format(now,time))
        f.close()
        print(response.text)
except Exception as e:
    print(e)

查看运行后的日志记录

测试完成。添加到定时任务中。
