weblogic后台密码解密

weblogic后台密码解密

使用一剑大佬已经写好的解密工具，不需要本地安装weblogic环境，工具已经自带lib依赖文件。成功率比较高，简单易用，比较推荐。

/u01/oracle/user_projects/domains/base_domain/security/SerializedSystemIni.dat

拿到SerializedSystemIni.dat文件后，使用工具weblogic_decrypt.jar进行直接解密。

起因

最近又出了新的漏洞，想着没事的时候复现一下。之前在windows平台搭建的weblogic环境竟然没有记录密码。真是做什么事都要有记录啊。

改密码真的麻烦，后台找到可以直接在本地解密weblogic后台密码的方法。

柳暗花明又一村啊。。。

先记录一下本次使用的方式。

直接在本地解密

<%@page pageEncoding="utf-8"%>
<%@page import="weblogic.security.internal.*,weblogic.security.internal.encryption.*"%>
<%
   EncryptionService es = null;
   ClearOrEncryptedService ces = null;
    String s = null;
    s= request.getParameter("pwd");
    es = SerializedSystemIni.getEncryptionService();
    if (es == null) {
       out.println("Unable to initialize encryption service");
        return;
    }
    ces = new ClearOrEncryptedService(es);
    if (s != null) {
        out.println("\nDecrypted Password is:" + ces.decrypt(s));
    }
%>

保存上述代码为getpwd.jsp文件。

将getpwd.jsp放到/u01/oracle/wlserver/server/lib/consoleapp/webapp/css目录中。

打开/u01/oracle/user_projects/domains/base_domain/config/config.xml文件

复制值{AES256}Oc7ue0fsbVuIWhA2hmS6R4Jn8rZrFkJ8e7I0m0XNNqQ=

然后访问：http://your-server-ip:7001/console/css/getpwd.jsp?pwd={AES256}Oc7ue0fsbVuIWhA2hmS6R4Jn8rZrFkJ8e7I0m0XNNqQ=

完美，就这样找回了后台密码。成功登录后台。

总结

以后做事情还是的记录过程。要不然就会像现在这样浪费时间。
